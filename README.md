SQI Agent


### Tech Stack

* Frontend: React (Vite) + Material UI
* Backend: Node.js + Express
* Language: JavaScript


### How to Run
-------------
Open two terminals and run code below in one of terminal

cd server
npm install
node server.js


This Will Start Backend Server

---------
Open Second terminals and run code below in that terminal

cd client
npm install
npm run dev

This Will Start Frontend Server
